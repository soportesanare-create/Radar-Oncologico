Política de privacidad operativa

  • Este radar es para observación comercial y derivación ética, no para crear bases de datos sensibles de pacientes.
  • No captures diagnóstico, estadio, mutaciones, pronóstico ni información clínica identificable desde publicaciones públicas.
  • Deriva a WhatsApp, formulario o llamada y solicita consentimiento antes de recabar datos personales sensibles.
  • Restringe tu API key de Google por dominio y por API, o muévela a backend para producción.
