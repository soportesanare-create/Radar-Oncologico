SANARÉ RADAR ONCOLÓGICO v3.2

Cambios:
- Se retiró la dependencia de Google Custom Search JSON API para el reporte diario.
- El módulo de reporte diario ahora consolida búsquedas hechas en el radar y observaciones guardadas el mismo día.
- Ya no requiere API key ni cx para generar el resumen diario y exportarlo a TXT.
- Mantiene observaciones locales, filtros, KPIs y exportación JSON.

Uso:
1) Descomprime la carpeta.
2) Abre index.html en tu navegador.
3) Haz tus búsquedas en el radar o abre Google desde los botones existentes.
4) Guarda observaciones con enlace, fuente, resumen y consulta origen.
5) En la sección Reporte diario, presiona "Generar reporte del día".
6) Exporta el TXT si lo necesitas.

Nota:
Para automatización real en segundo plano, el siguiente paso sería usar backend o una integración externa.
